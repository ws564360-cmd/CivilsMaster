    📚 Book Reading Tips | చదివే విధానం

    • ✅ Read one standard book completely before starting another.
    • ✅ Prepare short notes while reading.
    • ✅ Revise the same book multiple times.
    • ✅ Solve Previous Year Questions after each subject.
    • ✅ Read newspapers daily for Current Affairs.
    • ✅ Revise weak topics every week.
